# C 语言实现简单计算器

本课程用 C 语言做一个简单的计算器，进行加、减、乘、除操作。本程序涉及的所有数学知识都很简单，但输入过程会增加复杂性。都是 C 语言很基础的东西（输入输出），学了之后能写出一个完整的 C 语言程序。适合涉及 C 语言的基本语法，与输入与输出。适合初学者学习。

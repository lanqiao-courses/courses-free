# 字典类型

## 回忆

- 上次学习了字典
- 字典是用来查的
  - 根据一个 key
  - 可以查到相应的 value
- 字典中有很多 key-value 构成的键值对
  - 所有的键可以转化一个列表
  - 所有的值也可以转化一个列表

# jQuery 基础入门

jQuery 是一个优秀的轻量级 Javascript 库，兼容 CSS3，还兼容各种浏览器，在 Web 前端开发中广泛使用。课程将从 jQuery 基础入手，我们会学习到 jQuery 的具有鲜明特征的用法，例如 jQuery 选择器、jQuery 的事件和动画等，最终达到熟练掌握 jQuery 的目的。
